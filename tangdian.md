## 终端命令行
ls
查看目录下的文件和文件夹

ls -l
long选项，显示更多的信息

ls -a
显示隐藏文件和文件夹

cd 文件夹

change dir（切换工作目录）

cd ..
切换到父目录

cd
不带参数切换到Home目录

cd -
切换到之前的工作目录

pwd
print work dir（打印当前的工作目录）

mkdir
创建一个目录

mkdir -p
创建多层目录

rm
remove(删除普通文件）

rm -rf
删除文件夹及其下面的所有文件

cp

copy(拷贝文件)

cp -r
拷贝文件夹

mv
move重命名文件/文件夹或是移动文件/文件夹

touch
创建一个空文件
## brew（软件管理）

brew uninstall  //卸载软件

brew search     //查询可供安装的软件
